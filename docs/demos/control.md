- ControlZoom: leaflet 内置功能
- ControlAttribution: leaflet 内置功能
- ControlScale: leaflet 内置功能
- ControlFullscreen: 基于 leaflet-fullscreen
- ControlSearch: 搜索
- ControlLayers: leaflet 内置功能
