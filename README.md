# chaos-form-antd4

一个简单 antd4 表单控件适配器。

## ✨ Features

* Upload 文件上传。

## 📦 Install

```
npm i chaos-form-antd4 --save
```

## 🔨 Usage

```
import { Upload } from 'chaos-form-antd4';
```

## 🖥 Development

```
$ git clone https://github.com/Alfred-sg/chaos-form-antd4
$ npm install
$ npm run docs
```
打开浏览器访问 http://127.0.0.1:8000。
